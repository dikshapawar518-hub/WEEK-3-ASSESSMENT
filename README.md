# SQL Weekly Assessment 3 – Flipcart Products Dataset

## 📌 Overview
This repository contains my solutions for **Weekly Assessment 3** from The Unlox Academy. The assessment focuses on practicing fundamental SQL concepts using a fictional e-commerce dataset named **flipcart.products**.

## 🎯 Objectives
- Write SQL queries using SELECT, WHERE, ORDER BY, and LIMIT
- Apply filtering with IN, BETWEEN, and LIKE
- Use DISTINCT and Aggregate Functions
- Perform data grouping with GROUP BY
- Filter grouped data using HAVING
- Handle NULL values using IS NULL and COALESCE/CASE
- Implement conditional logic using CASE WHEN

## 🛠️ Technologies Used
- MySQL
- MySQL Workbench

## 📂 Files Included
- `flipcart_setup.sql` – Database creation and sample dataset
- `Week3_Assessment.sql` – SQL solutions for all assessment questions

## 📚 Topics Covered
- Basic SQL Queries
- Filtering Records
- Sorting Data
- Aggregate Functions
- GROUP BY & HAVING
- NULL Handling
- CASE WHEN
- LIMIT
- DISTINCT

## 📊 Dataset Information
The assessment uses a fictional **Flipcart Products** dataset containing:
- 30 Products
- 10 Columns
- Categories such as Electronics, Apparel, Home, Books, Beauty, and Sports
